# Truffle minimal template

### Installation
Node: 18.15.0

NPM: 9.3.1
```bash
npm install -g truffle
```
### Installation dependencies
```bash
npm install
```
### Compile contracts
```bash
truffle compile
```
### Run migrations
```bash
truffle migrate --network goerli
```
### Verify your contract
```bash
truffle run verify Token --network goerli
```
